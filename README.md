# LLM-TeacherEval

![Python](https://img.shields.io/badge/python-3.12-blue)

LLM-TeacherEval is a research-focused framework for evaluating large language models on English language education tasks. It is designed to support benchmarking, quality evaluation, hallucination detection, and report generation for educational use cases.

## Overview

This project aims to provide a structured evaluation pipeline for LLMs in English language teaching and learning contexts. It focuses on educational quality, safety, and explainability so reviewers can compare models and identify strengths and weaknesses.

## Features

- Benchmark multiple LLMs across English education tasks
- Evaluate pedagogical quality and teaching effectiveness
- Detect hallucinations and factual inconsistencies
- Assess safety and content appropriateness
- Support human review and annotation workflows
- Generate structured evaluation reports
